Data query

You can use this page to search for an Old Manunian who was killed in WW1, using data you know. As you enter more data into the fields below, the table will become smaller, showing only relevant records held. Once you have found the table row for the record you would like to view in more detail, click the blue "View this record" button in that row.

Each record is given a siteIndex number. This is only for the functionality of this website, and is not held anywhere else. You can, however, enter a siteIndex number to quickly access a record you have previously found.

For information about how to interpret the data you see here, please visit the Understand the data page.

If you cannot find a record, please check the data you have entered, especially regarding spelling. If a record is missing, please contact the MGS Archives Department - details are given on the About page.
