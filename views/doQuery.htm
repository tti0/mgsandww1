Data query

You can use this page to search for an Old Manunian who was killed in World War One, using data you know.

Click me for more detailed information about the Data query page

Below is a table listing every record we hold. You may need to use the left-right scroll bar at the bottom of the table to view all of the rows. You can filter the records by:

  • Clicking on the 'hamburger' () icon next to the field you want to filter, for example, "Age at death";
  • Entering a value to be filtered, for example, "20";
  • The table will only show records whose Age at death was 20.

By clicking on a row, you will be taken to a page giving full details and images (where avaliable) for the record you clicked on.

Each record is given a siteIndex number. This is only for the functionality of this website, and is not held anywhere else. You can, however, enter a siteIndex number to quickly access a record you have previously found.

If you cannot find a record, please check the data you have entered, especially regarding spelling. If you believe that a record is missing, please contact the MGS Archives Department - details are given on the About page.
