Data query

You can use this page to search for an Old Manunian who was killed in World War One, using data you know.

Each record is given a siteIndex number. This is only for the functionality of this website, and is not held anywhere else. You can, however, enter a siteIndex number to quickly access a record you have previously found.

If you cannot find a record, please check the data you have entered, especially regarding spelling. If a record is missing, please contact the MGS Archives Department - details are given on the About page.

Click a row to view the record.
