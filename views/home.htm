Welcome to the MGS and WW1 website

This is a website with searchable details of Old Mancunians who died in the First World War, and details of the school's links to the war more generally.

To learn more about Manchester Grammar's links to the First World War, visit the In memoriam and Primary sources, sections, which contain static resources written by the MGS Archives and History Department.

To find an Old Mancunian who faught in the war, please visit the Data query page, and enter data you already know to find an Old Mancunian.

To find out about this website, who made it, and how it was made, please visit the About page.
