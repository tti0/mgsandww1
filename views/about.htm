About

This website was developed to allow spectators of the Manchester Grammar School's annual Senior Sports Day to view results, events, and competitors in these events. It is a collaborative effort between the Computer Science and PE departments of the school, the staff responsible in each department being Mr D Millington and Mr T Grainger respectively. It is hoped that students, parents and staff will find this website beneficial. It works by fetching score and competitor data from a Google Drive spreadsheet, and displaying it in an easy-to-use and clear fashion.

How it works

This website is powered by AngularJS and Google Sheets. We query a Google Sheet with lots of live data in it every time a page is loaded, using an XMLHttpRequest();. (For the curious, the full, public Google Sheet is avaliable here: https://goo.gl/kut8PQ.) The Google Sheets API is somewhat difficult to work with, so some "de-Google-ification" needs to take place to the data returned. We then use various algorithms to compile the data into neatly laid-out tables, cross-referencing each other for easy navigation. The website is styled by Bootstrap. It's made to be as compact as possible so that it loads quickly on phones and doesn't use up 'too much data'. It's entirely responsive, and was tested mobile-first.
